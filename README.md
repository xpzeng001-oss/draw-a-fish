# 🐟 一起画鱼 (Draw a Fish)

一款受 drawafish.com 启发的微信小程序，让用户画鱼、看鱼游泳、PK 评分！

## 功能特性

- 🎨 **自由画鱼** - 画布支持多颜色、多画笔大小、橡皮擦、撤销
- 🐠 **鱼缸动画** - 矢量鱼在 Canvas 2D 鱼缸中实时游泳，支持物理碰撞和尾巴摆动
- 📊 **AI 评分** - 基于画作覆盖率、形状比例、颜色丰富度的智能评分系统
- 🏆 **排行榜** - 按分数排名，展示所有鱼的信息
- 👆 **互动** - 点击鱼缸中的鱼会吓跑它们
- 🫧 **沉浸体验** - 气泡、水草、光线等海底氛围特效

## 使用方法

1. 用**微信开发者工具**打开项目目录
2. AppID 可使用测试号，或填写你自己的小程序 AppID
3. 点击编译即可预览

## 项目结构

```
draw-a-fish/
├── app.js/json/wxss        # 全局配置
├── pages/
│   ├── index/              # 首页 - 鱼缸主界面
│   ├── draw/               # 画鱼页面
│   └── tank/               # 排行榜
├── utils/
│   └── fish-engine.js      # 鱼的绘制与动画引擎
└── project.config.json     # 开发者工具配置
```

## 自定义

- 修改 `app.js` 中的 `_generateSampleFish()` 自定义初始鱼群
- 修改 `fish-engine.js` 中的 `drawVectorFish()` 自定义鱼的外观
- 修改 `fish-engine.js` 中的 `evaluateFishScore()` 调整评分逻辑
